Commands

Tab

m

c

open or close the "commands modal".

q

close the "more modal" or the "commands modal".

s

focus search input.

Ctrl + s

on search input, returns to the previous focus if exist.

r

on item, remove the item.

w

on item, remove the complete day range.

on item, opens the item in a contrary way that the open behavior

on item, opens the item in a contrary way that the open behavior

on item, opens the item in a new windows

on item, opens the item in a new windows
